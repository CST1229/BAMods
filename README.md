## Other Mods

> [!CAUTION]
> These haven't been updated since November 2025. Expect them to be broken in the current BA version. I'm procrastinating too much to bother updating them.

(By the way, you can extract any of these to view their source code)

- [BALib](https://cst1229.eu.org/BAMods/mods/CST1229-BALib.zip) - library mod for adding tiles, etc.

- [BADimensions](https://cst1229.eu.org/BAMods/mods/CST1229-BADimensions.zip) - dimension-switching ala Super Mario Bros. Dimensions
- [crateslop](https://cst1229.eu.org/BAMods/mods/CST1229-crateslop.zip) - enables experimental mode
- [Cutscene Voice Acting](https://cst1229.eu.org/BAMods/mods/CST1229-CutsceneVoiceActing.zip) - joke mod that pronounces cutscenes phonetically
- [Mini Mode](https://cst1229.eu.org/BAMods/mods/CST1229-MiniMode.zip) - adds mini rays. broken in the current version
- [Structures in Favorites](https://cst1229.eu.org/BAMods/mods/CST1229-StructuresInFavorites.zip) - put structures in your favorites via Ctrl+(click Stamp tool)
